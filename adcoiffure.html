A&D Coiffure du Monde

Site web développé pour une entreprise de coiffure basée aux USA.

Description du projet

Le site web A&D Coiffure du Monde a été développé pour une entreprise de coiffure basée aux États-Unis. Le site présente les services offerts, les tarifs, et permet aux clients de prendre rendez-vous en ligne.

Technologies utilisées

  • HTML, CSS, JavaScript
